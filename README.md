# Crest AI 🤖

> AI-powered Telegram assistant for crypto, Web3, blockchain analytics, and smart conversations.

---

## ✨ Features

- 🤖 AI Chat Assistant
- 💹 Crypto & Web3 Knowledge
- ⚡ Fast AI Responses
- 📡 Telegram Bot Integration
- 🧠 Powered by Groq AI
- 🛠 Lightweight Node.js Architecture

## 🚀 Commands

```bash
/start
/help
/ping
/ai <prompt>
```
